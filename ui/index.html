    Glossary

    what the numbers mean
    GT
    Ground Truth — human-labeled text + polygon from labelme JSON.
    PR
    Predicted — what the OCR engine returned.
    F1
    Harmonic mean of detection precision and recall. ↑ 1.0 perfect
    CER
    Character Error Rate — edit distance / GT length. ↓ 0.0 perfect
    WER
    Word Error Rate — same as CER at word level. ↓ 0.0 perfect
    IoU
    Intersection over Union — polygon overlap. ≥ 0.5 = match. ↑ 1.0 identical
    TP / FP / FN
    Matched / extra / missed detection lines.
    Conf
    Average OCR confidence on matched lines.
